### 👨🏽‍💻 Sobre Mim

Sou **Analista e Desenvolvedor de Sistemas** com um forte direcionamento para a arquitetura de software, boas práticas de engenharia e resolução de problemas estruturais. Tenho ampla experiência na construção de sistemas robustos, desde a concepção do design de banco de dados até as interfaces no frontend.

- 🏗️ **Arquitetura & Clean Code**: Foco absoluto na escrita de um código limpo, testável e manutenível.
- 🚀 **Performance**: Busco sempre a otimização de consultas, fluxos de API e renderização no client.
- 📚 **Aprendizado Contínuo**: Acredito que a tecnologia está sempre evoluindo e busco me atualizar ativamente.
- 🗿 *"A persistência é o caminho do êxito."*
